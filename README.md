# Python Log Analyzer

A beginner cybersecurity project written in Python that analyzes Linux authentication logs.

## Features

- Counts successful logins
- Counts failed logins
- Tracks login attempts by username
- Tracks login attempts by IP address

## Skills Practiced

- Python functions
- File handling
- Lists
- Loops
- Conditional statements
- collections.Counter
- Modular project structure

## Future Improvements

- Return values instead of printing directly
- Detect brute-force attacks
- Export reports to a text file
- Accept custom log files as input
